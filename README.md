# Dose+
一个希望减轻长期用药者心智负担的应用。

- Flutter + Meterial You，我喜欢多彩的界面。
- 本地存储数据，我省事你放心。
- 代码由 AI 生成，以及不会 Flutter 的本人粗暴修改，质量不高。
- 初始用户调研仅两人，欢迎反馈各种想法。
